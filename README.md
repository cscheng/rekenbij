# Rekenbij

Een interactieve webaplicatie voor kinderen in groep 3 en 4 om te oefenen met plussommen, minsommen en tafels. Het is ontworpen met een eenvoudige opzet zonder overbodige visuele afleidingen, zodat kinderen zich beter kunnen concentreren op het rekenen.

[Live Demo](https://cscheng.github.io/rekenbij)

## Development

Gebouwd met [Astro](https://astro.build), [Typescript](https://www.typescriptlang.org/) en [React](https://react.dev/):

```sh
npm install
npm run dev
```
